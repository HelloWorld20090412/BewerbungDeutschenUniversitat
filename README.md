# 留德申请资料导航

这是一个可直接部署到 GitHub Pages 的静态网站。

## 使用方式

1. 新建一个 GitHub 仓库。
2. 把本文件夹里的 `index.html`、`styles.css`、`script.js` 和 `assets` 文件夹上传到仓库根目录。
3. 在 GitHub 仓库的 Settings -> Pages 里，选择从 `main` 分支部署。
4. 等 Pages 生成链接后，就可以把网址发到群里。

## 修改内容

- 改链接：打开 `index.html`，搜索原链接后替换。
- 改颜色或排版：修改 `styles.css`。
- 改复制和搜索功能：修改 `script.js`。
